# Network Anomaly Detection

A machine learning project for detecting network intrusions using the CICIDS2017 dataset.

## Dataset
Download the CICIDS2017 dataset from: https://www.unb.ca/cic/datasets/ids-2017.html  
Place the CSV files in the `data/` folder.

## Project Structure
- `data/` — dataset files (not tracked by git)
- `notebooks/` — Jupyter notebooks for EDA and modelling
- `models/` — saved trained models
- `dashboard/` — Streamlit app
- `docs/` — reference documents
